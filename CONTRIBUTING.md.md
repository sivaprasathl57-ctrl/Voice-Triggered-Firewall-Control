## Getting Started

1. Fork this repository and clone it:
   ```bash
   https://github.com/sivaprasathl57-ctrl/Voice-Triggered-Firewall-Control.git

   cd voice-firewall
   ```
2.Create a Python virtual environment:
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
3.Run the project:
```bash
python3 main.py
```
